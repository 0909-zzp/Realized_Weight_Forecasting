# VARX 网格搜索规范 — 参数调优

## 1. 目标

从 Model 1 开始逐层叠加特征块和正则化项，在**验证集**上确定各参数的最优值，使 Model 4 (Network VARX) 的验证集 MSE 显著低于 Model 3 (Sparse VARX)，同时 Model 5 (Network+Smooth) 的换手率不超过 Model 4 的 1.5 倍。

| 指标 | 目标 | 判定方式 |
|------|------|------|
| Model 4 val MSE < Model 3 val MSE | 网络特征必须提供增量信息 | 验证集 MSE 对比 |
| Model 5 换手率 ≤ 1.5× Model 4 | 平滑不能过度牺牲精度 | 验证集日换手率 |
| 系数稀疏度 ≥ 50% (Model 2+) | Lasso 有效压缩 | `mean(\|β\| < 1e-8) ≥ 0.5` |
| 网络掩码密度 15%~40% | 有区分度但不过度稀疏 | 度 60~150 |

## 2. 参数范围

论文公式(13)定义 **4 个独立超参数** + 网络阈值 τ，共 **5 个可调参数**：

| 论文符号 | 代码参数 | 默认值 | 作用 |
|:--:|:--:|:--:|------|
| λ₁ | `LAMBDA_LASSO` | 1e-4 | 连接资产的滞后系数 ℓ1 |
| λ₂ | `LAMBDA_LASSO + LAMBDA_NETWORK` | 1.1e-3 | 未连接资产的滞后系数 ℓ1（λ₂ > λ₁） |
| λ₃ | `LAMBDA_EXOG` | 1e-4 | 外生变量系数 ℓ1（独立于 λ₁） |
| λ_s | `LAMBDA_TURNOVER` | 1e-2 | 换手率平滑 ℓ2 |
| τ | `NETWORK_THRESHOLD` | 0.7 | Ā 截断阈值 |

按搜索轮次分三组：

| 轮次 | 参数 | 默认值 | 扫描值 | 依赖 |
|:--:|------|:--:|------|------|
| 1 | `LAMBDA_LASSO` | 1e-4 | `[1e-5, 5e-5, 1e-4, 5e-4, 1e-3]` | — |
| 1 | `LAMBDA_EXOG` | 1e-4 | `[1e-5, 5e-5, 1e-4, 5e-4, 1e-3]` | — |
| 2 | `NETWORK_THRESHOLD` | 0.7 | `[0.5, 0.7, 0.8]` | λ₁, λ₃ 锁定 |
| 2 | `LAMBDA_NETWORK` | 1e-3 | `[0, 1e-4, 1e-3, 1e-2]` | λ₁, λ₃ 锁定 |
| 3 | `LAMBDA_TURNOVER` | 1e-2 | `[0, 1e-4, 1e-3, 5e-3, 1e-2]` | λ₁, λ₃, τ, λ₂ 锁定 |

> **固定参数**：`Ā_ROLLING_WINDOW = 20` 不做搜索（GLasso 阶段已定）。

## 4. 分阶段流程

### 模型参数消融逻辑

各模型从低级到高级逐层叠加。承继前提：同特征集。

| # | 模型 | λ₁ | λ₂ | λ₃ | λ_s | τ | 自环豁免 | 承继规则 |
|:--:|------|:--:|:--:|:--:|:--:|:--:|:--:|------|
| 2 | Sparse VAR | ✅ | — | — | — | — | — | 独立搜索 |
| 3 | Sparse VARX | ✅ | — | ✅ | — | — | — | 独立搜索 λ₁×λ₃ |
| 4 | Network VARX | ✅ | ✅ | ✅ | — | ✅ | ✅ | 独立搜索 λ₁×τ×λ_net |
| 5 | +Smooth | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | 继承M4, 搜索λ_s |
| 6 | DFL (L2闭式) | — | — | — | — | — | — | 后处理, η/ρ/DL_box调参 |

### 最终参数 (2026-07-15)

| 参数 | 值 |
|---|:---:|
| λ₁ (M3/M4/M5) | 3e-4 |
| λ₁ (M2独立) | 5e-4 |
| λ₃ | 5e-4 |
| τ | 0.7 |
| λ_net | 1e-3 |
| λ_s | 1e-3 |
| ETA | 1e-4 |
| RHO_DFL | 1e-3 |
| DFL_BOX | 0.05 |

### DFL 注记
DFL 采用 L2 闭式解 (非网格搜索调参): w* = (Sigma+(eta+rho)I)^(-1).(eta.w_prev+rho.w_stat)
Sigma 为训练期末40天滚动协方差。盒约束 +/-5%。

### 4.1 第 0 轮：M1 基准 + M2 参数搜索

**模型**：Model 1 (VAR, OLS) — M2 (Sparse VAR)

**原因**：确立预测下限。M1 无参数（OLS），M2 仅滞后块 + λ₁ Lasso。

**扫描空间**：`LAMBDA_LASSO`，5 组（M2 仅滞后块，与 M3 特征维度不同，不可继承）
- `LAMBDA_LASSO ∈ {1e-5, 5e-5, 1e-4, 5e-4, 1e-3}`

**选择标准**：M1 固定；M2 取验证集 MSE 最小（无需稀疏度约束，M2 无后续网络惩罚依赖）

**输出**：`best_λ₁_M2`（仅供参考，M3 须独立搜索）

### 4.2 第 1 轮：基础稀疏结构 + 外生惩罚

**模型**：Model 3 (Sparse VARX)，滞后权重 + 外生变量 + Lasso

**原因**：M3 与 M2 特征维度不同（lagged vs lagged+exog），λ₁ 须在自己的特征集上独立搜索。λ₁ 和 λ₃ 分别控制滞后权重和外生变量的稀疏度。

**扫描空间**：`LAMBDA_LASSO × LAMBDA_EXOG`，4 × 4 = 16 组
- `LAMBDA_LASSO ∈ {5e-5, 1e-4, 5e-4, 1e-3}`
- `LAMBDA_EXOG ∈ {5e-5, 1e-4, 5e-4, 1e-3}`

**选择标准**：验证集 MSE 最小，且稀疏度 ≤ 92%。（**若仅评估 M3 本身性能，纯贪心 MSE 即可；此处需要对比 M3 vs M4，必须加入稀疏度约束**。M4 网络惩罚的设计前提是"存在可被网络结构区分的交叉资产效应"——如果 λ₁ 过大导致稀疏 ≥99%，交叉项几乎全零，M4 的网络惩罚无靶子可罚，此时 M3 vs M4 实质上退化为"Lasso vs Lasso+自环豁免"，不是公平的消融实验。）

**输出**：锁定 `LAMBDA_LASSO`, `LAMBDA_EXOG`

### 4.4 第 3 轮：网络正则化（λ₂ × τ）

**模型**：Model 4 (Network VARX)。M3 和 M4 共用同一 x_t，λ₁ 和 λ₃ **直接继承 R1**。R2 只调网络惩罚独有参数。

**原因**：x_t 相同 → Lasso 全局稀疏度不变 → λ₁/λ₃ 无需重调。

**扫描空间**：`τ × λ₂`，6 × 5 = 30 组

| 参数 | 扫描值 | 说明 |
|------|------|------|
| τ | `{0.5, 0.6, 0.7, 0.8, 0.85, 0.9}` | 网络邻域阈值 |
| λ₂ | `{λ₁, 5e-4, 1e-3, 5e-3, 1e-2}` | 未连接滞后 L1（λ₂=λ₁→无惩罚, 1e-2→最强） |

**选择标准**：验证集 MSE 最小。vs 同 λ₁/λ₃ 的 Model 3 对比。

**输出**：锁定 `NETWORK_THRESHOLD`, `LAMBDA_NETWORK`

### 4.5 第 3 轮：换手率平滑

**模型**：Model 5 (Network+Smooth)，Model 4 + 数据增广 λ_s 惩罚

**原因**：λ_s 越大换手率越低但 MSE 越高，需要在验证集上找平衡点。此行独立于前两轮（不改变网络结构）。

**扫描空间**：`LAMBDA_TURNOVER ∈ {0, 1e-4, 1e-3, 5e-3, 1e-2}`（5 组，λ₁, λ₃, τ, λ₂ 均已锁定）

**选择标准**：验证集 MSE 最小，且换手率 ≤ 1.5× Model 4 的验证集换手率

**输出**：锁定 `LAMBDA_TURNOVER`

### 4.6 最终评估

用 4.1-4.5 五轮搜索锁定的全部最优参数，替换 `共享模块.py` 中的对应值，然后运行 `VARX预测.py` 的完整 7 模型流水线（train→test），产出最终的 Table2。

## 6. 验收标准

### 6.1 搜索阶段（验证集）

| 检查项 | 要求 |
|---|---|
| 最优 λ₁, λ₃ 不在边界 | 5e-5 ~ 5e-4 范围内（非边缘值） |
| 最优 τ 产生 15%~50% 密度 | 避免全连接或过度稀疏 |
| 最优 λ_net > 0 | 网络正则化非退化（≠0 表示产生了效果） |
| 第 2 轮 Model 4 MSE < 同 λ_lasso 下 Model 3 MSE | 网络特征在最优参数下提供增量信息 |

### 6.2 最终评估（测试集）

| 检查项 | 要求 | 方法 |
|:--:|:--:|------|
| **DM 检验** | Model 4-6 vs Model 1 显著 | `dm_test()` p < 0.05 |
| **MSE** | Model 4 < Model 3 | 测试集 MSE |
| **换手率** | Model 5 ≤ 1.5× Model 4 | 测试集日换手率 |
| **稀疏度** | ≥ 50% | Lasso 截断比例 |
| **PSI** | 报告但不卡（≤ 0.5 通过率报告） | 分位数分箱 |

### 6.3 退化回退

若搜索后 Model 4 仍无法超越 Model 3，按规范 §5.4 回退：

```
Network VARX 无效 → 回退到 Sparse VARX（删网络特征块）
Smooth 过度牺牲精度 → λ_turn = 0（关闭平滑）
均无效 → 以 Sparse VAR 为最终模型
```
