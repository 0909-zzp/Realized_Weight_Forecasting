## 特征工程修复（3个Bug）

### Bug 1: Ā展平导致维度爆炸
- **现象**：X_features.npy = (2415, 156025), 2.88 GB
- **根因**：`n_a = K*K` 将392×392=153,664维Ā展平混入X
- **修复**：Ā独立保存为 A_bar.npy (2415,392,392)，供VARX构建正则化掩码M
- **结果**：X → (2415, 2361), ~40 MB, 维度从156K降至2.4K

### Bug 2: PSI用density=True导致数值爆炸
- **现象**：psi_max=38,953, psi_mean=762
- **根因**：density=True返回密度值而非比例，窄bin下密度极大
- **修复**：改用分位数分箱+比例（标准PSI公式）
- **结果**：psi_max=12.79, psi_mean=0.60, psi_p50=0.35, psi_pass_0.5=60.3%

### Bug 3: VIX全零（类型不匹配）
- **现象**：exog_0 (VIX) std=0，全部特征=0
- **根因**：vix_daily.csv index为int64，load_exogenous用str(d)查找永远False
- **修复**：`pd.read_csv(..., dtype={0: str})` + `index.astype(str)`
- **结果**：n_zero_var_features: 1→0, VIX数据恢复正常

## VARX预测.py Model 5 平滑改为训练时实现

### 问题
原 Model 5 用后处理指数平滑（SMOOTH_ALPHA=0.7），不影响系数估计，与论文公式13的 λ_s 不一致，学术严谨性不足。

### 修复：数据增广法
- 删除 `smooth_predictions()` 后处理函数
- 在 `fit_model()` 中：若 `cfg['smooth']=True`，追加 √λ_s·ΔX 行（目标=0）
- 数学等价：½‖y-Xβ‖² + ½λ_s‖ΔX·β‖² + α‖β‖₁ = 原始Lasso + λ_s平滑惩罚
- `LAMBDA_S = LAMBDA_TURNOVER = 1e-2`
- 规范文档新增 §2.3.3 说明此方法

### 审查发现8个问题
1. 路径错误（OUT_DIR/code/输出数据 → 应为 特征工程/）
2. 网络掩码写了但没用（注释"简化实现"）
3. 只有1个模型（规范要求5个）
4. 无MSE/MAE/DM/MCS指标
5. import不存在的模块
