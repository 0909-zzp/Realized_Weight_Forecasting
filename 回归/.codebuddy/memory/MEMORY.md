# 课程论文项目 - 长期记忆

## 项目概述
- 现代回归分析课程论文：数字经济与区域经济差距的实证分析
- 使用 PKU-DFI 指数 + 省域面板数据（2011-2022年）

## 文件说明
- `现代回归分析课程论文(1)(1).docx` — 论文正文（前四章已写，缺实证结果）
- `课程论文提纲.docx` — 课程要求大纲（需含R软件、描述统计、结果、附录）
- `regression_analysis.py` — Python 完整实证代码（模拟数据）
- `panel_data.csv` — 生成的面板数据（30省×12年）
- `desc_statistics.csv` — 描述性统计结果
- `results_summary.csv` — 回归结果汇总（6个模型）

## 技术选型
- 用户要求 Python（非R），因为实际数据只有 DFI 指数
- 面板回归：linearmodels.PanelOLS（个体固定效应，entity_effects=True）
  - 注意：不用 time_effects=True（双向FE会吸收DFI的时间变异）
- IV估计：手动两步法（Phase1: PanelOLS 预测 ln_DFI → Phase2: PanelOLS 回归 Gap）
- 可视化：matplotlib + seaborn（中文标签会报警告，但图片可正常生成）
